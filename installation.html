Installation¶

The mrsimulator package requires the ‘fftw3’ package to function properly. Install the fftw3 package using

>>> conda install -c eumetsat fftw3

Read more about fftw3 installation.

To install the mrsimulator package, first download package and run the following in the terminal

>>> pip install git+https://github.com/DeepanshS/mrsimulator.git@master
