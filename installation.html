Installation¶

Installing requirements¶

The mrsimulator package requires the fftw3 C routines to function. Download and install the fftw3 by typing the following in the terminal

conda install -c eumetsat fftw3

Additionally, mrsimulator requires the following packages.

  • NumPy>=1.13.3 for array manipulation.
  • astropy>=3.0 for units library.
  • mkl, mkl-include for lineshape calculation.

For figures and visualization, we use

  • matplotlib>=3.0.2.

For the web-face interface, we use

  • plotly>=3.6
  • dash>=0.40
  • dash_daq>=0.1

Installing mrsimulator¶

To install the mrsimulator package, type the following in the terminal.

pip install git+https://github.com/DeepanshS/mrsimulator.git@master

Test¶

If the installation is successful, you should be able to run the following in the terminal.

python -c "import mrsimulator; mrsimulator.run_test()"
